# Chat

###Install

```
git clone https://github.com/chenshuchao/socketio_cpp.git
cd socketio_cpp
mkdir build
cd build
cmake ..
make
```

###Run

```
./bin/chat ../example/chat
```

浏览器访问 http://127.0.0.1:5011/index.html (请使用 chrome 等高级浏览器）
